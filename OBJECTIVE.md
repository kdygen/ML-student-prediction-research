# OBJECTIVE.md

# Research Goal

Develop accurate and reproducible early prediction models for student outcomes using OULAD.

The objective is not simply to maximize predictive performance, but to understand which information available early in a course is most useful for prediction.

---

## Current Direction

Current milestone:

Develop an early prediction pipeline using only information available before the chosen cutoff.

---

## Optimization Priorities

Priority order:

1. Prevent data leakage
2. Improve predictive performance
3. Improve feature quality
4. Improve reproducibility
5. Improve code organization

---

## Current Focus

Current work should prioritize:

- early engagement features
- temporal behavioral features
- assessment-based features available before cutoff
- robust evaluation

---

## Experimental Principles

Every experiment should test a hypothesis.

Avoid random feature creation.

New features should have an explanation for why they may improve prediction.

---

## Success Criteria

Prefer improvements that are:

- reproducible
- interpretable
- robust across validation
- available before prediction time

Small improvements supported by evidence are preferred over large unexplained gains.
